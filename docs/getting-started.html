    Getting Started with ChatPharo

    Welcome to the ChatPharo documentation! This guide will help you get started with installing and using ChatPharo in your Pharo environment.

    Installation

    To install ChatPharo, follow these steps:

    1. Open your Pharo image.
    2. Open a Playground and execute the following command:
      3. Metacello new
